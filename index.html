¿Cómo puedo hacer animaciones de matemáticas?

After Effects

  • Es un software de Adobe que sirve para realizar animaciones.
  • Es muy potente, pero tiene una curva de aprendizaje muy grande.
  • Pese a su potencia, para realizar animaciones de matemáticas, se requiere automatizar muchas cosas.
  • Ello requiere scripting, sin embargo, en After Effects posee un scripting muy básico.
  • Pese a lo anterior, sería muy útil para un proceso de posproducción.
  • Es muy caro, y no es software libre.

Blender

  • Es un software de modelado y animación 3D.
  • Es software libre, y tiene una comunidad muy grande.
  • Es muy potente, pero tiene una curva de aprendizaje muy grande.
  • Es muy útil para realizar animaciones de matemáticas, pero requiere automatizar muchas cosas.
  • Ello requiere scripting, y Blender posee un scripting muy potente.
  • Es muy útil para un proceso de posproducción.

Morpholib

  • Es una librería de Python de animaciones en general.
  • Es software libre, pero no tiene una comunidad muy grande.
  • No es tan potente como Manim, y su curva de aprendizaje es muy grande.
  • Es útil para realizar animaciones de matemáticas, y al estar en un lenguaje de programación, facilita la producción.
  • Es útil para la producción.

Reanimate

  • Es una librería de Haskell de animaciones en general.
  • Es software libre, pero no tiene una comunidad muy grande.
  • No es tan potente como Manim, y su curva de aprendizaje es muy grande.
  • Es útil para realizar animaciones de matemáticas, y al estar en un lenguaje de programación, facilita la producción.
  • Es útil para la producción.

Manim

  • Es una librería de Python de animaciones de matemáticas.
  • Es software libre, y tiene una comunidad muy grande.
  • Es muy potente, y su curva de aprendizaje es muy grande.
  • Es muy útil para realizar animaciones de matemáticas, y al estar en un lenguaje de programación, facilita la producción.
  • Es muy útil para la producción.

¿Qué es Manim?

  • Manim es una librería de Python para crear animaciones de matemáticas.
  • Fue creado por Grant Sanderson, más conocido como 3Blue1Brown.
  • Este es el software que Grant usa para hacer sus videos.
  • El 2020 un grupo de desarrolladores de la comunidad de Manim hizo su propia edición, distinta del original (en ese entonces el cairo-backend).
  • Grant Sanderson hizo una nueva versión de Manim, la cual es desde el 2021 la original, llamada ManimGL.

Versiones de Manim

  • Manim 3b1b cairo-backend: versión anteriormente original de 3b1b que usó para videos antiguos, su forma de renderizar videos es usar Cairo y FFmpeg.
  • ManimGL: versión original de 3b1b que usa actualmente para hacer sus videos, su forma de renderizar videos es usar OpenGL y FFmpeg.
  • ManimCE: versión comunitaria de Manim que es la más usada por otros usuarios para sus videos. Tiene una excelente documentación y responde rápidamente a cambios que desee la comunidad.

¿Cómo funciona Manim?

  • Manim usa Python 3.7 o superior para que el usuario pueda escribir sus escenas.
  • Manim usa LaTeX para renderizar texto matemático.
  • Manim usa OpenGL o Cairo para hacer los fotogramas.
  • Manim usa FFmpeg para renderizar videos a partir de los fotogramas.
